This blog explains a breakdown of what I am learning on Security Architecture

## Network Topology

```
Internet → Firewall → Router → Switch → Workstations & Servers  
                                  ↘ Wi-Fi Access Points  
                                  ↘ Printers, IoT Devices

```

## IP Addressing
- Router (Gateway): 192.168.1.1

- Firewall: 192.168.1.2

- Servers: 192.168.1.10 - 192.168.1.20

- Workstations: 192.168.1.50 - 192.168.1.100

- Printers/IoT: 192.168.1.200 - 192.168.1.250

- Wi-Fi VLAN: 192.168.2.0/24
